# 제곱 ㄴㄴ 수

### Gold 1

어떤 정수 X가 1보다 큰 제곱수로 나누어 떨어지지 않을 때, 그 수를 제곱ㄴㄴ수라고 한다. 제곱수는 정수의 제곱이다. min과 max가 주어지면, min보다 크거나 같고, max보다 작거나 같은 제곱ㄴㄴ수가 몇 개 있는지 출력한다.

## 입력
첫째 줄에 두 정수 min과 max가 주어진다.

## 출력
첫째 줄에 min보다 크거나 같고, max보다 작거나 같은 제곱ㄴㄴ수의 개수를 출력한다.

## 제한
- 1 ≤ min ≤ 1,000,000,000,000
- min ≤ max ≤ min + 1,000,000

## 문제풀이
굉장히 큰 수가 들어갈 수 있기 때문에 순회 범위를 잘 조절해야 하는 문제이다. 제곱수가 약수인지 파악하는 문제이기 때문에 `max_value`값의 제곱근까지 순회하며 약수를 골라낸다.

이와 동시에 소수의 제곱수의 배수를 결과 값에서 제외한다. 이를 위해 `res`라는 변수에 `min_value`에서 `max_value`까지에 해당하는 1로 가득찬 리스트를 할당한다. 그리고 `range`를 통해 제외시킨 뒤 `res`안 1의 개수를 세서 결과값을 출력한다.
